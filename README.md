# ファイルタイムスタンプエディタ

Windows PowerShell GUIでファイルのタイムスタンプを編集するツール。

## 機能

- 作成日時の編集
- 更新日時の編集
- アクセス日時の編集
- ドラッグ&ドロップでファイル指定
- 参照ボタンからファイル選択

## 使い方

1. `FileTimestampEditor.ps1` を右クリックして「PowerShellで実行」を選択
2. ファイルをウィンドウにドラッグ&ドロップ、または「参照...」ボタンでファイルを選択
3. 表示されたタイムスタンプを編集
4. 「適用」ボタンをクリックして保存

## 動作環境

- Windows PowerShell 5.1以降
- .NET Framework 4.5以降
